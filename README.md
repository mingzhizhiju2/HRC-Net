# HRC-Net
HRC-Net repo (to be released): PyTorch implementation of hierarchical radius convolution for point cloud classification/segmentation. Includes HRConv operator, multi-feature input (9D), channel attention, training scripts for ModelNet40/ScanObjectNN/ShapeNet Part, and pre-trained models. Lightweight (0.79M params, 0.77G FLOPs).
